AI Chatbot (Real-Time)

This is a simple real-time AI chatbot built using React, WebSocket, and Node.js.
It supports live streaming responses, typing indicator, and dark mode.

The goal of this project is to understand real-time communication and how AI responses can be streamed like ChatGPT.

------- Features

- Real-time chat using WebSocket

- AI response streaming (token by token)

- Typing indicator while AI is thinking

- Auto scroll to latest message

- Dark / Light mode

- Copy message text

- Messages stored in localStorage

-------- Tech Stack

- Frontend: React + TypeScript + Tailwind CSS

- Backend: Node.js + WebSocket

AI: Gemini / Groq (server side)
